## Installation

To run this app locally, follow these steps:

1. Clone the repo locally

```bash
git clone https://github.com/danielkiing3/custom_content_scrollbar.git
cd custom_content_scrollbar
```
2. Install dependencies:
```bash
flutter pub get
```
3. Run the app:
```bash
flutter run
```
